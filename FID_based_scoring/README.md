# Frechet Inception Distance

### Formula Used:
    FID = (mu_1 - mu_2)^2 + Tr[C_1 + C_2 - 2 * sqrt(C_1 * C_2)]


### Creat environment:
    python -m venv FID
    FID/Scripts/activate

### Install libraries: 
    cd FID_based_scoring
    pip install -r requirements.txt

### Run the file
    python main.py
